Security in Real-time Cyber Physical Systems

Nanyang Technological University (NTU), Singapore (Summer 2017)

  • It is possible to model Cyber-Physical systems at different levels of abstraction. However, it is very complex to analyse the different system-level vulnerabilities without detailed prior knowledge of the system being analysed. Thus, it is important that security analysis be performed at the initial stages of deployment and that newer tools and techniques be explored to perform security analysis for already existing Cyber-Physical Systems. This project aims at making use of a heterogenous modeling tools Metro II, to model the notorious Stuxnet attack.
  • Keywords : Cyber-Physical Systems, Security, Modeling, Stuxnet
  • Code Report
